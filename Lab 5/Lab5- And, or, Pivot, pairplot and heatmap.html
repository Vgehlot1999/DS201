We will use the same baseball data set as in HW5. Here are the details for your reference. This dataset contains data concerning a baseball team's performance in a given year. It has the following variables:

Team: A code for the name of the team

League: The Major League Baseball league the team belongs to, either AL (American League) or NL (National League)

Year: The year of the corresponding record

RA: The number of runs allowed by the team in that year

W: The number of regular season wins by the team in that year

SLG: The slugging percentage of the team in that year

BA: The batting average of the team in that year

Playoffs: Whether the team made the playoffs in that year (1 for yes, 0 for no)

RankSeason: Among the playoff teams in that year, the ranking of their regular season records (1 is best)

RankPlayoffs: Among the playoff teams in that year, how well they fared in the playoffs. The team winning the World Series gets a RankPlayoffs of 1.

G: The number of games a team played in that year

OOBP: The team's opponents' on-base percentage in that year

OSLG: The team's opponents' slugging percentage in that year

Boolean Operators in python¶

Boolean is a type of value that can be either True or False. In Python, the Boolean type is bool, which is a subtype of int. Python has mainly 2 Boolean operators that are typed out as plain English words:

  1. or

  2. and

These operators connect Boolean expressions (and objects) to create compound Boolean expressions.

The Python Boolean operators always take two Boolean expressions or two objects or a combination of them, so theyre considered binary operators.

In this lab, we will be using these in the context of pandas dataframe.

OR operator¶

When I write a or b, the statement "a or b" is TRUE in all of the following situations except the last:

  • if a is true regradless of b being true or false.

  • if b is true regardless of a being true or false.

  • if both a and b are true

  • "a or b" is false only when both a and b are false.

Let's apply "or" in our baseball data frame. The notation for the "or" operator is | (called pipe or vertical bar) sign. The set notation equivalent for "or" is the union of two sets...... i.e {1,3} | {2,3} = {1,2,3}.

Pivot Table in Pandas.¶

A pivot table is a data processing tool used to query, organize, and summarize data or information between spreadsheets, tables, or databases.

The generic pivot table term is used by many companies and software vendors. PivotTable, Microsoft branded and trademarked version, is used in most Excel products. We will use PivotTable in pandas. A pivot table is useful when analyzing a large amount of data, as it allows users to apply specific criteria to summarize, organize and reorganize data tables, and create reports.

For example, when a store manager reviews the sales of a specific item over a six-month period, he must sift through many pages of relevant and irrelevant data. A pivot table, however, simplifies this process by automatically counting, summarizing, and sorting the data. Summarized data may then be used to create a report tailored to the item's activity within the specified six-month parameter, reducing data output and table rows and columns. This process may be used to accommodate various data requirements.

Pairwise plot and Heatmap: Pairs plot is the scatterplot of every two pair of variables in the data. A correlation heatmap uses colored cells, typically in a monochromatic scale, to show a 2D correlation matrix (table) between two discrete dimensions or event types. The values of the first dimension appear as rows of the table, while the values of the second dimension are represented by the columns of the table. The color value of the cells is proportional to the number of measurements that match the dimensional values. This enables you to quickly identify incidence patterns, and to recognize anomalies. Heatmap is a very popular graphing technique. You see it everywhere from weather data analysis to web page analytics.
